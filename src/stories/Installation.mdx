> Nemeton Digital System is a React component library

## Getting Started

To install Nemeton Digital System (@nds_core) library, run the following command.

@nds_core/nds currently requires React 18.x or higher.

<Canvas> npm install @nds_core/nds</Canvas>

OR

<Canvas> yarn install @nds_core/nds</Canvas>
